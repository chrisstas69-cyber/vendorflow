# VendorFlow OS

Event vendor intelligence for street fair and festival operators — deadline alerts, financial scoring, and tier ranking via Airtable.

## What's in this repo

| File | Purpose |
|------|---------|
| `vendorflow-os-airtable.js` | Main Airtable scripting extension (deadlines, finance, scoring) |
| `file3_airtable_deadline_engine.js` | Deadline alert engine |
| `file4_vendorflow_engine.py` | Python scoring / analysis engine |
| `VENDORFLOW-SETUP.md` | Airtable table setup + automation guide |
| `FIELD_CONTRACT.md` | Field definitions and data contract |
| `files/` | Setup docs + demo video |
