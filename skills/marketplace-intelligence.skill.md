# Skill: Marketplace Intelligence

## Context
Aggregating supply (vendors) and demand (organizers) into a functional, searchable marketplace.

## Step-by-step plan
1. Create unified search index for events and vendors.
2. Implement geo-spatial filtering for event discovery.
3. Build category-based matching engine to suggest vendors to organizers.
4. Add marketplace analytics dashboard for system-wide overview.
5. Develop public-facing marketplace landing page for visitor discovery.
6. Refactor existing discovery logic to use the new unified search service.
7. Verify all marketplace components and summarize implementation.
